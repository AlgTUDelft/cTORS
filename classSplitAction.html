const int SplitAction::GetSplitIndex ( ) const
inline

Get the split index of the split.

A ShuntingUnit consisting of Trains A-B-C-D will split as follow with split index:

  1. SU1: A, SU2: B-C-D
  2. SU1: A-B, SU2: C-D
  3. SU1: A-B-C, SU2: D
