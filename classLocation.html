Detailed Description

A Location describes a shunting yard.

It contains a list of Tracks and Facilities. It also has helper functions to find (shortest) Paths and distances

inline

Get all the neighboring paths from a certain position.

A neighboring path is a path to a neighboring Railroad track. Every track has immediate neighbors as defined by the A-side and B-side attributes. This method returns all (non-immediate) Railroad neighbors that can be reached by traversing zero or more non-Railroad tracks.

Call CalcNeighboringPaths (once) to calculate all the neighboring paths
