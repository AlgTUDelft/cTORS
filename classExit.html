Construct an Exit action for the ShuntingUnit described by the train ids and the id of the Outgoing event.

The ShuntingUnit attribute of the Outgoing event may refer to unspecified trains, that is trains of some type, with no set id. Therefore the train ids of the ShuntingUnit in the shunting yard need to be specified seperately.
